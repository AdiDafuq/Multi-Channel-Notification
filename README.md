# JobTracker Backend

A Spring Boot backend application for managing job applications with AI-powered features.  
Built with Java, Spring Boot, PostgreSQL, and Redis for scalability and resilience.

---

## 🚀 Features
- **Job Application Management**
  - Create, Read, Update, Delete (CRUD) job applications
- **AI Integration**
  - Summarize job descriptions
  - Generate tailored cover letters
- **Exception Handling**
  - Global error handling with custom responses
- **Validation**
  - DTO-based request validation
- **Persistence**
  - PostgreSQL database with JPA/Hibernate
- **Caching & Guardrails**
  - Redis for rate limiting, cooldowns, and request throttling
- **Scalable Design**
  - Stateless APIs, horizontal scaling, Dockerized infrastructure

---

## 🛠️ Tech Stack
- **Language:** Java 17+
- **Frameworks:** Spring Boot, Spring Data JPA, Spring Web
- **Database:** PostgreSQL (Dockerized)
- **Cache/Guardrails:** Redis
- **Build Tool:** Maven
- **Testing:** JUnit
- **Infrastructure:** Docker Compose
